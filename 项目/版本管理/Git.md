# 常见分支类型
main/master   --主分支
develop  -- 开发分支
feature -- 功能分支
hotfix  -- 修复分支

# 常用的git命令
## git version --查看git版本
## git branch  --列出分支列表
## git branch + 分支名 --创建一个新分支但是不会自动切换
## git checkout + 分支名 --切换到某一个分支
## git switch +分支名 --切换到某一个分支
## git checkout -b +分支名  --创建并切换分支
## git branch -d + 分支名  --删除分支 （这里d是安全删除的意思，只有分支的改变完全合并了才允许删除） 分支删除但是提交历史保存



# 工作流程
1、修改代码
2、添加到暂存
3、提交更改
4、推送到远程

# 合并分支
1、切换到目标分支，一般为主分支main
> git switch main

2、合并特定分支
> git merge +分支名


# 解决合并冲突
## 冲突发生的原因
> 同一文件，同一位置，不同修改

## 解决步骤
1、查看冲突文件
2、手动编辑解决
3、添加到暂存区
4、完成合并提交
